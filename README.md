# drawingApp

MeteorJS & MongoDB - Responsive Web Development Course - Collaborative Web Application - Live Drawing

## Local running of the app

Open Terminal navigate to the project folder and run:

```
$ meteor
```

## Built With

* Meteor.js
* MongoDB
* d3.js
